Overview

CpGenius is a web tool for feature selection and processing of methylation data. This documentation provides guides for developers and users to operate, extend and deploy the application.

Getting Started

Quick start instructions:

  1. Install project dependencies from `pyproject.toml`.
  2. Run the FastAPI dev server: uv run uvicorn app.start_fastapi:app --reload.
  3. Open the browser at http://localhost:8001/ and navigate to features.

API

API endpoints and usage:

  • /fs/upload — Upload a CSV for feature selection.
  • /fs/run-algorithm — Trigger feature selection run.
  • See docs/FILE_UPLOAD_API.md and docs/PROGNOSIS_API.md for detailed examples.
